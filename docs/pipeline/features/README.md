# Features — Casa fiecărui feature din pipeline

> Fiecare feature care intră în pipeline primește un folder aici,
> cu toate artefactele sale: design, ticket-uri, implementare.

---

## Convenție

```
docs/pipeline/features/<nume-feature>/
├── FEATURE_DESIGN.md       ← Specificație (Game Design)
├── DESIGN_PASS.md          ← Mecanici detaliate (Game Design)
├── UI_WIREFRAME.md         ← Wireframe-uri (dacă e cazul)
├── SHARED_IMPLEMENTATION_GUIDE.md  ← Pattern-uri specifice (Production)
└── tickets/
    ├── TICKET_INDEX.md     ← Toate ticket-urile feature-ului
    ├── T###-nume-ticket.md ← Ticket individual
    └── ...
```
